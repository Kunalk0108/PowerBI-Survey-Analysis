# Power BI Survey Analysis

## Overview
This Power BI project analyzes survey data to uncover key insights and trends. The dashboard presents interactive visualizations to help users explore the dataset effectively.

## Files
- **Project Survey.pbix** - The Power BI report file containing the visualizations and analysis.
- **Power BI - Final Project.xlsx** - The dataset used for building the report.

## Features
- **Data Cleaning & Transformation**: The dataset was processed using Power Query.
- **Interactive Visualizations**: The report includes filters, slicers, and drill-downs for better exploration.
- **Key Insights**: Identifies trends based on survey responses.

## How to Use
1. Download both the `.pbix` and `.xlsx` files.
2. Open `Project Survey.pbix` in Microsoft Power BI Desktop.
3. Ensure the dataset (`.xlsx` file) is correctly linked.
4. Explore the dashboards and insights.

## Tools Used
- **Microsoft Power BI** for visualization
- **Excel (Power Query)** for data preprocessing
